# AGENTS.md — Awesome Vector Databases

## Purpose

This repository is a curated registry of vector databases, vector search engines, and embedding storage systems.

## Scope

Cover resources for storing and searching vector embeddings at scale:

- Purpose-built vector databases.
- Traditional databases with vector extensions.
- Vector search libraries and approximate-nearest-neighbor (ANN) indexes.
- Managed vector-search services.

## Style

- Follow the same format as the other `awesome-*` repositories in the Enterprise AI Atlas organization.
- Label official/vendor resources as `Official` and third-party resources as `Community`.
- Prefer official documentation and links.

## Maintenance

- Run `./scripts/validate-links.sh` before finishing any edit.
- Place entries under the deployment model or database category they primarily target.

## Important reminders

- Do not provide legal advice.
- Verify that links resolve and are current.
